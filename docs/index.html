Why BareMetal?

Architecture principles.

layers_clear

Zero Dependencies

No NPM hell. No node_modules black hole. Just pure, native JavaScript executing directly in the V8 engine as intended.

lock_open

No Vendor Lock-In

Your code is your code. BareMetal augments standard DOM APIs rather than wrapping them in proprietary synthetic events.

speed

Unapologetic Performance

Bypass the Virtual DOM overhead. Direct surgical DOM updates yield 60fps animations and near-zero latency interactions.

Core Engine Features

Systems level capabilities.

0ms Hover Pre-fetching

Anticipates user intent. Assets and JSON payloads are aggressively fetched on mouse-enter, guaranteeing instantaneous perceived load times on click.

Smart DOM Keep-Alive

Inactive views are detached and cached in memory, not destroyed. Returning to a previous route restores scroll position and state immediately.

State Hydration

A reactive signals-based store that automatically binds to DOM nodes. Update the state, and only the bound text nodes or attributes mutate.

Component Lazy Loading

Define components as async closures. BareMetal orchestrates the dynamic import of CSS and JS only when the component enters the viewport.
